Predatory API

This API returns a list of journals as described on the “Beall’s List of predatory journals and publishers” found at https://beallslist.weebly.com/.

The API uses webtask.io and is rate limited at one request per second, and does not require any authentication. If you need a higher read volume, please cache or download the data.

The API only accepts requests over https.

Basics

To retrieve the full data set (~400kb), perform a GET on https://wt-ece6cabd401b68e3fc2743969a9c99f0-0.run.webtask.io/predatory

Search Queries

You can also add a search query to the end of the url with ?search=. To search for Frontiers in Microbiology, add: ?search=frontiers%20in%20microbiology to get a filtered set of results. Adding a query will also return a list of matches against DOAJ’s database.
